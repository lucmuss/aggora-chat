# Phase 06 API And Polish

## Ziel

Agora erweitert die Plattform um produktnahe Schnittstellen und Betreiber-Workflows:

- oeffentliche REST-API
- Mod Mail
- Community-Settings
- Image-Posts
- Feed-Caching

## Scope

### API

- Popular-Feed
- Community-Feed
- Post-Detail
- Kommentar-Listing
- Post-Create
- Comment-Create
- Vote
- Search
- User-Profile und User-Posts

### Moderation

- einfache Mod-Mail-Threads
- Replies durch Mods und Nutzer

### Community Management

- Settings fuer Posttypen und Vote-Hide-Window
- Removal-Reason-Verwaltung

### Performance

- erster Cache fuer Popular- und Community-Feeds

## Akzeptanzkriterien

- die wichtigsten JSON-Endpunkte sind nutzbar
- Mod Mail laeuft als Thread-System
- Owner oder Mods koennen Community-Settings aendern
- Image-Posts lassen sich erstellen
- Feed-Caching greift fuer Popular- und Community-Ansichten
