## Ziel

Die Dokumente sollen als Referenz fuer den naechsten Implementierungsplan dienen.

- Produktmodell: Reddit-aehnliche Community-Plattform
- Identitaet: pseudonym-first
- Moderation: agentenunterstuetzt, queue-first, auditierbar
- Technikrichtung: Django + HTMX + Tailwind, serverseitig priorisiert
